Per avviare il progetto TestFabrik è necessario creare un database Postgres contocorrente
Non è necessario lanciare alcuno script per la creazione delle tabelle dato che è presente la seguente property nel file application.properties impostata su update 
spring.jpa.hibernate.ddl-auto=update

allego comuque lo script sql per la creazione del DB e della tabella.

E' necessario impostare nel file application.properties le credenziali del database corrette.
